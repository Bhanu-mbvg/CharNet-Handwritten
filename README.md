# CharNet-Handwritten

Character Convolution Network to recognize the digits and characters from A-Z and a-z

## Features
| Component | Accuracy |
| --- | ----------- |
| Digits (0-9) | ~98% |
| Characters (A-Z) | ~91% |
| Characters (a-z) | ~89% |
